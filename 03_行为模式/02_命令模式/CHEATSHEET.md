## 关键参与者

| 类 | 职责 | 备注 |
|----|------|------|
| `CookerZeff` | 接收者，真正做菜的人 | 包含 4 个做菜方法 |
| `AbstractCommand` | 抽象命令类 | `excute()` + `name()` |
| `DSXCommand` 等 | 具体菜品命令 | 调用 CookerZeff 对应方法 |
| `WaiterLuffy` | 调用者 | map<int, list> 存储订单 |

## 调用流程

```
顾客 -> WaiterLuffy.setOrder(桌号, 命令) -> 加入订单列表
顾客 -> WaiterLuffy.cancelOrder(桌号, 命令) -> 从订单列表移除
WaiterLuffy.notify(桌号) -> 遍历命令 -> 每个命令.excute() -> CookerZeff.具体方法()
WaiterLuffy.checkOut(桌号) -> 列出该桌所有菜品
```

## 何时使用

- 需要将操作对象化、参数化
- 需要队列、日志、撤销/重做功能
- GUI 按钮与具体操作解耦
- 需要对操作进行排队或延迟执行

## 编译与运行

```bash
make
./main
```
