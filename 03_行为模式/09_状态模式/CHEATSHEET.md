## 状态转换图

```
  上午(Forenoon)
    │  time >= 11
    ▼
  中午(Noon)
    │  time >= 13
    ▼
  下午(Afternoon)
    │  time >= 18
    ▼
  晚上(Evening)
    │  time >= 19
    ▼
  睡觉(结束)
```

## 关键参与者

| 类 | 角色 | 时间范围 | 行为 |
|----|------|----------|------|
| `AbstractState` | 抽象状态 | -- | `working(Sanji*)` |
| `ForenoonState` | 上午状态 | < 11 | 早餐/钓鱼 |
| `NoonState` | 中午状态 | < 13 | 做午饭 |
| `AfternoonState` | 下午状态 | < 18 | 甜点/钓鱼 |
| `EveningState` | 晚上状态 | >= 18 | 晚饭/睡觉 |
| `Sanji` | 上下文 | -- | 持有时间+状态 |

## 何时使用

- 对象行为随内部状态变化
- 大量 if-else 判断状态
- 状态数量有限但频繁切换
- 订单状态机、游戏角色、工作流

## 编译与运行

```bash
make
./main
```
