# 设计模式 (Design Patterns)

基于 C++17 的 24 个设计模式可运行示例项目，使用《海贼王》角色和场景作为示例。

> 原笔记来源：[苏丙榅 - 设计模式](https://subingwen.cn/design-patterns/#1-%E5%89%8D%E8%A8%80)
>
> 整合作者：[ol木子李lo](https://1613661434.github.io/)

## 快速开始

### 命令行

```bash
# 进入任一模式目录
cd "01_创建型模式/01_单例模式"

# 编译 + 运行
make run

# 清理
make clean
```

### VSCode Task

1. `Ctrl+Shift+P` → `Tasks: Run Task`
2. 选择 `▶ 单例模式 (Singleton)` 或任意其他模式
3. 自动编译并运行，结果显示在终端

任务列表：
- `▶ 模式名` — 编译 + 运行单个模式（24 个）
- `build-all` — 一次编译全部 24 个模式
- `clean-all` — 清理全部编译产物
